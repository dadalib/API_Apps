## Getting Started

### Prerequisites

* Pillow
  ```
  pip  install Pillow
  ```
  ...

### Installation

1. Get a free API Key at [https://developer.ilovepdf.com/login](https://developer.ilovepdf.com/login)

2. Clone the repo
   ```sh
   git clone git@github.com:dadalib/api_apps.git
   ```
3. Install  packages
   ```sh
   pip install -r requirements.txt
   ```
